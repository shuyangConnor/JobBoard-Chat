The Github repository for the Job Board example is at

github.com/graphql-by-example/job-board

As shown in the initial Github Repository video, you can use it as a reference if you have any problems with your own code while following along.

The repository contains a commit for each lecture, with the message matching the lecture name.  You'll find a Code Diff link in the Resources for each lecture, taking you to the right commit page on Github that shows the changes made in that video.

From the commit page, you can also download the code at that specific point by clicking on the Browse files button on the top-right corner, then select Code > Download ZIP.
