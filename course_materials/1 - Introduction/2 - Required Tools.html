To follow along with the code examples in this course you will need:

  • A computer running Linux, macOS, or Windows.

  • A web browser, preferably the latest version of Google Chrome since I use its Developer Tools extensively in the videos.

  • A recent version of Node.js installed. The latest LTS (Long Term Support) version is recommended, i.e. currently v20.x.

  • A code editor, preferably Visual Studio Code since that's what I use in the videos.

    • If you do use VS Code, the only extension I recommend pre-installing is ESLint.
